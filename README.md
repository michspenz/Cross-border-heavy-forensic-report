### Challenge Overview
Reconstruct the complete supply chain, operational timeline, convoy structure, and co-conspirators from vehicle telematics data (navigation.db, edr_events.db, phone_sync.db, system_log.db, NMEA streams, etc.).

### Key Findings
- 11 successful smuggling runs between Tema Industrial Area (Warehouse B4) and Aflao border
- Consistent return-leg overload detected via suspension logs and DTC C1A00
- Multi-vehicle convoy coordination via Bluetooth
- Post-interdiction panic deletion attempt and recovered SMS ("Delete everything")
